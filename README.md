# URL Shortener

## Problem Statement
Long URLs are inconvenient to share. This project generates short, unique URL codes using hashing.

## Features
- Hash-based URL shortening
- Persistent storage using JSON
- URL retrieval

## Tech Stack
- Python
- Hashing
- JSON

## How to Run
python shortener.py

## Future Improvements
- Collision handling
- Web interface
- Expiry-based URLs
